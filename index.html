Timepicki

A simple jQuery based time picker control plugin

Download - more details

Timepicki is a simple and free time picker jQuery plugin, it has been tested to run on jQuery 1.5 onwards

It has been designed to be mobile/tablet friendly, disabling the on-screen keyboard when it is opened and improved touch handling

It has been tested on an iPad, Android (built in browser and Chrome) and of course desktop browsers
