Timepickr

A simple jQuery based time picker control plugin

Download

Timepickr is a simple and free time picker jQuery plugin, it has been tested to run on jQuery 1.5 onwards up to 2.1 (currently)

It has been designed to be mobile/tablet friendly, disabling the on-screen keyboard when it is opened and having improved touch handling. It is also designed to be completely inclusive with jQuery being its only dependency.

It has been tested on an iPad, Android (built in browser and Chrome) and of course desktop browsers (Firefox, Safari, Internet Explorer and Chrome)

Developed by anami
